# Tarea 1 — ¿Qué es una base de datos?

## Objetivo
Diseñar en papel o en este archivo la estructura de una tabla para un caso simple.

## Instrucciones
1. Elige uno de estos casos (o uno propio si prefieres):
   - Lista de estudiantes de tu salón
   - Inventario de una tienda pequeña
   - Agenda de contactos
2. Define una tabla con al menos 5 columnas.
3. Para cada columna, indica su tipo de dato: texto, número entero, número decimal, fecha, etc.
4. Escribe 3 filas de ejemplo con datos inventados.
