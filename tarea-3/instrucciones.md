# Tarea 3 — SQL básico

## Objetivo
Instalar DB Browser for SQLite, crear tu propia base de datos y ejecutar consultas básicas.

## Paso 1: Instalar la herramienta
Descarga DB Browser for SQLite desde: https://sqlitebrowser.org/dl/

## Paso 2: Crear tu base de datos
1. Abre DB Browser for SQLite.
2. Toca "New Database" y guárdala como `mi_base.db` dentro de esta misma carpeta `tarea-3/`.
3. Crea la tabla que diseñaste en la Tarea 1 (misma estructura de columnas).
4. Inserta al menos 5 filas de datos.

## Paso 3: Escribir las consultas
Abre el archivo `entrega.sql` de esta carpeta y completa cada consulta pedida.

## Entregable
Sube dos archivos a esta carpeta:
- `mi_base.db` (tu base de datos)
- `entrega.sql` (completado, ver plantilla)
