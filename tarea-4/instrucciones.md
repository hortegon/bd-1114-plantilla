# Tarea 4 — SQL intermedio

## Objetivo
Practicar JOIN, funciones de agregación (COUNT, SUM, AVG) y GROUP BY.

## Paso 1: Generar la base de datos de ejemplo
1. Abre DB Browser for SQLite.
2. Crea una nueva base de datos llamada `ejemplo.db` en esta carpeta `tarea-4/`.
3. Ve a la pestaña "Execute SQL".
4. Copia y pega todo el contenido del archivo `datos_ejemplo.sql` de esta carpeta y ejecútalo (esto crea las tablas `estudiantes` y `calificaciones` con datos de prueba).

## Paso 2: Escribir las consultas
Abre `entrega.sql` de esta carpeta y completa cada consulta.

## Entregable
Sube dos archivos a esta carpeta:
- `ejemplo.db` (generada en el paso 1)
- `entrega.sql` (completado)
