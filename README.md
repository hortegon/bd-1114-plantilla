# Proyecto de Bases de Datos - Grupo 1114

Este repositorio contiene las 5 tareas del módulo de bases de datos. Cada tarea está en su propia carpeta con instrucciones.

## Cómo empezar

1. Crea tu copia de este repositorio con el botón verde "Use this template" (arriba de la lista de archivos).
2. Trabaja las tareas en orden, dentro de tu propia copia.
3. Cada tarea indica dónde trabajas y dónde entregas: algunas se entregan como **formulario (Issues)** y otras subiendo **archivos** (bases de datos SQLite).

## Tareas

| Tarea | Tema | Instrucciones | Cómo se entrega |
|---|---|---|---|
| 1 | ¿Qué es una base de datos? | [`tarea-1/`](./tarea-1) | Formulario (pestaña Issues) |
| 2 | Modelo relacional | [`tarea-2/`](./tarea-2) | Formulario (pestaña Issues) |
| 3 | SQL básico | [`tarea-3/`](./tarea-3) | Archivos subidos a la carpeta |
| 4 | SQL intermedio | [`tarea-4/`](./tarea-4) | Archivos subidos a la carpeta |
| 5 | Conexión con proyecto real | [`tarea-5/`](./tarea-5) | Formulario (pestaña Issues) |

## Cómo usar el formulario de una tarea (Tareas 1, 2 y 5)

1. Toca la pestaña **Issues** de tu repo (arriba, junto a "Code").
2. Toca **New Issue**.
3. Elige la plantilla de la tarea correspondiente.
4. Completa los campos del formulario.
5. Toca **Submit new issue**. Listo, esa es tu entrega.

## Herramienta para SQL (Tareas 3 y 4)

Usaremos **DB Browser for SQLite** (gratis, sin instalación de servidor): https://sqlitebrowser.org/dl/
