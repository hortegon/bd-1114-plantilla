# Proyecto de Bases de Datos - Grupo 1114

Este repositorio contiene las 5 tareas del módulo de bases de datos. Cada tarea está en su propia carpeta con instrucciones y archivos de partida.

## Cómo empezar

1. Crea tu copia de este repositorio con el botón verde "Use this template" (arriba de la lista de archivos).
2. Trabaja las tareas en orden, dentro de tu propia copia.
3. Cada tarea indica qué archivo debes completar y subir (commit).

## Tareas

| Tarea | Tema | Carpeta |
|---|---|---|
| 1 | ¿Qué es una base de datos? | [`tarea-1/`](./tarea-1) |
| 2 | Modelo relacional | [`tarea-2/`](./tarea-2) |
| 3 | SQL básico | [`tarea-3/`](./tarea-3) |
| 4 | SQL intermedio | [`tarea-4/`](./tarea-4) |
| 5 | Conexión con proyecto real | [`tarea-5/`](./tarea-5) |

## Herramienta

Usaremos **DB Browser for SQLite** (gratis, sin instalación de servidor): https://sqlitebrowser.org/dl/
